Where does this data come from?

Every figure on this page is transcribed directly from the Township of West Windsor's adopted 2026 municipal budget, as filed with the New Jersey Division of Local Government Services. Original department labels, FCOA? codes, and source-sheet references are preserved for every row so you can look any number up in the source document yourself.

Does this page change or recalculate any numbers?

No. A few rules keep every figure faithful to the source workbook:

  • Values are copied from the source workbook without recalculation.
  • Blank fields in the original budget are left blank here too — never replaced with zero.
  • Spending totals shown in charts exclude the form's internal subtotal/total rows to avoid double-counting; the full Data Tables page includes them, clearly labeled.
What about school and county taxes?

This page covers only the municipal budget — the township government's own operations (police, roads, parks, etc.). School funding is a separate budget set by the West Windsor-Plainsboro Regional School District, and it's the largest part of most residents' property tax bills: school taxes for 2025 totaled $111,688,841, roughly twice the size of this entire municipal budget. County taxes are set separately by Mercer County and aren't included here either.

Is this an official Township publication?

No. This tool is built and maintained by West Windsor Forward, a nonpartisan civic organization. It is not an official Township of West Windsor publication and does not replace the adopted budget document — it's meant to make that document easier to understand.

Can I get the raw data myself?

Yes — every table behind this page is searchable on the Data Tables page, or you can download the underlying data as JSON and explore it yourself.
